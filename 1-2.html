02-1 숫자형

숫자형으로 할 수 있는 것들

  • 사칙연산

    • >>> a = 3
    >>> b = 4
    >>> a + b
    7
    >>> a * b
    12
    >>> a / b
    0.75

    *는 곱하기, /는 나누기

  • 제곱

    • >>> a = 3
    >>> b = 4
    >>> a ** b
    81

    a의 b승 이므로 3의 4승은 81

  • % 연산자

    • >>> 7 % 3
    1
    >>> 3 % 7
    3

    7을 3으로 나눈 나머지니깐 1, 3을 7으로 나눈 나머지니깐 3

  • // 연산자

    • >>> 7 / 4
    1.75
    >>> 7 // 4
    1

    7을 4로 나눈 몫이니깐 1


    메인화면으로 가기
