02-3 리스트 자료형

  • 1, 3, 5, 7, 9라는 숫자 모음


    • >>> odd = [1, 3, 5, 7, 9]

    리스트는 변수를 한번에 관리한다

  • 리스트명 = [요소1, 요소2, 요소3, ...]


    • >>> a = [ ]

    >>> b = [1, 2, 3]

    >>> c = ['Life', 'is', 'too', 'short']

    >>> d = [1, 2, 'Life', 'is']

    >>> e = [1, 2, ['Life', 'is']]

    리스트는 변수를 한번에 관리한다

  • 리스트의 인덱싱


    • >>> a = [1, 2, 3]

    >>> a[0]

    1

    >>> a[0] + a[2]

    4

    >>> a[-1]

    3

    -는 뒤에서부터

  • 리스트의 슬라이싱


    • >>> a = [1, 2, 3, 4, 5]

    >>> a[0:2]

    [1, 2]

    >>> b = a[:2]

    >>> c = a[2:]

    >>> b

    [1, 2]

    >>> c

    [3, 4, 5]

  • 리스트 더하기


    • >>> a = [1, 2, 3]

    >>> b = [4, 5, 6]

    >>> a + b

    [1, 2, 3, 4, 5, 6]

  • 리스트 반복하기


    • >>> a = [1, 2, 3]

    >>> a * 3

    [1, 2, 3, 1, 2, 3, 1, 2, 3]

  • 리스트에서 하나의 값 수정하기


    • >>> a = [1, 2, 3]

    >>>a[2] = 4

    >>> a

    [1, 2, 4]

  • 리스트에서 연속된 범위의 값 수정하기


    • >>> a = [1, 2, 3]

    >>> a[1:2]

    [2]

    >>> a[1:2] = ['a', 'b', 'c']

    >>> a

    [1, 'a', 'b', 'c', 3]

  • [ ] 사용해 리스트 요소 삭제하기


    • >>> a = [1, 'a', 'b', 'c', 4]

    >>> a[1:3] = [ ]

    >>> a

    [1, 'c', 4]

    a[1:3] = [ ]에서 [ 와 ] 사이에 띄어쓰기는 못알아볼까봐 표시하려고 한거지 실제론 띄어쓰기 안씀

  • del 함수 사용해 리스트 요소 삭제하기


    • >>> a = [1, 'a', 'b', 'c', 4]

    >>> a[1:3] = [ ]

    >>> a

    [1, 'c', 4]

    >>> del a[1]

    >>> a

    [1, 4]

  • 리스트에 요소 추가(append)


    • >>> a = [1, 2, 3]

    >>> a.append(4)

    >>> a

    [1, 2, 3, 4]

    변수 a의 리스트에 4라는 요소를 추가한다

  • 리스트 정렬(sort)


    • >>> a = [1, 4, 3, 2]

    >>> a.sort()

    >>> a

    [1, 2, 3, 4]

    말그대로 리스트를 정렬하는거다

  • 리스트 뒤집기(reverse)


    • >>> a = ['a', 'c', 'b']

    >>> a.reverse()

    >>> a

    ['b', 'c', 'a']

    말그대로 리스트를 뒤집는거다

  • 위치 반환(index)


    • >>> a = [1, 2, 3]

    >>> a.index(3)

    2

    >>> a.index(1)

    0

    위치 찾는 명령어인데 a.index(3)에서 3이라는 숫자가 어디에 위치해 있는지 알려준다

  • 리스트에 요소 삽입(insert)


    • >>> a = [1, 2, 3]

    >>> a.insert[0, 4]

    [4, 1, 2, 3]

    변수 a의 리스트의 0번째 위치에 4라는 요소를 삽입한다

  • 리스트 요소 제거(remove)


    • >>> a = [1, 2, 3, 1, 2, 3]

    >>> a.remove(3)

    [1, 2, 1, 2, 3]

    변수 a의 리스트에 있는 3이라는 요소를 제거한다 (첫번째거만 지워지나봄)

  • 리스트 요소 끄집어내기(pop)


    • >>> a = [1, 2, 3]

    >>> a.pop()

    3

    >>> a

    [1, 2]

    뒤에서부터 하나씩 끄집어낸다

  • 리스트에 포함된 요소 x의 개수 세기(count)


    • >>> a = [1, 2, 3, 1]

    >>> a.count(1)

    2

    a.count(1)에서 1이라는 값의 개수를 센다

  • 리스트 확장(extend)


    • >>> a = [1, 2, 3]

    >>> a.extend([4, 5])

    >>> a

    [1, 2, 3, 4, 5]

    >>> b = [6, 7]

    >>> a.extend(b)

    >>> a

    [1, 2, 3, 4, 5, 6, 7]

    a.extend([4, 5])에서 변수a의 리스트에 4, 5의 리스트를 추가한다


    메인화면으로 가기
